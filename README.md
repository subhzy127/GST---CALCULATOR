GST Calculator

This is a simple GST Calculator created using HTML.

Features

- Enter product price
- Enter GST percentage
- Calculates GST amount
- Shows total price including GST

How to Use ?

1. Download or open the project folder
2. Double click index.html
3. Enter price and GST %
4. Click Calculate

The calculator will instantly show GST amount and total price.

Purpose

This project was created to practice basic HTML and build a simple business-related calculator useful for GST calculations.
